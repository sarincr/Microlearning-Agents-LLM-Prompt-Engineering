# 🧠 Microlearning: Agents, LLMs & Prompt Engineering

This repository is a structured collection of notes, experiments, and implementations focused on modern **Large Language Models (LLMs)** and **agent-based systems**.

It is designed as a hands-on learning space covering the progression from foundational concepts to production-level AI systems.

---

## 📦 What This Repository Contains

### 🔹 LLM Fundamentals
- Core concepts behind large language models
- Transformer architecture and attention
- Tokenization and embeddings
- Evolution of NLP → LLM systems

---

### 🔹 Prompt Engineering
- Techniques for interacting with LLMs effectively
- Zero-shot and few-shot prompting
- Chain-of-thought reasoning
- Structured outputs and prompt design patterns

---

### 🔹 Context & Memory Handling
- Managing context windows
- Chunking and retrieval strategies
- Embeddings and vector databases
- Short-term and long-term memory approaches

---

### 🔹 LangChain
- Building LLM-powered applications
- Chains, tools, and prompt templates
- Output parsing and memory integration

---

### 🔹 LangGraph
- Graph-based orchestration of LLM workflows
- Stateful agents and multi-step reasoning
- Designing cyclic and conditional flows

---

### 🔹 Multi-Agent Systems
- Architectures involving multiple collaborating agents
- Planner–executor patterns
- Role-based agent systems (researcher, writer, critic)

---

### 🔹 LangSmith
- Observability and debugging for LLM apps
- Tracing execution flows
- Evaluating prompts and system performance

---

### 🔹 RAG (Retrieval-Augmented Generation)
- Integrating external knowledge with LLMs
- Document processing and embedding pipelines
- Vector search and context injection

---

### 🔹 Fine-Tuning
- Customizing models for domain-specific tasks
- Instruction tuning and dataset preparation
- Efficient fine-tuning approaches (LoRA, PEFT)

---

## 🎯 Purpose

- Consolidate learning in LLMs and AI agents
- Build practical, real-world systems
- Document experiments and evolving understanding
- Serve as a reference for future projects
